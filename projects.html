Pro-tekt is a 24/7 hour GPS tracking system, Pro-tekt is a small device that is hard wired into your vehicles battery, once it is wired in you download the app and you can start tracking instantly. The device has a host of features, Including, Live tracking, Current location, Route history, Find my ride, a Geo fence and an alarm.

My self and one other programmer have been responsible for the entire project, which was created for a client, Matt Penhall. You can visit Pro-tekts site here

Pro-tekt is coded in Python and Javascript using the Python framework Flask.
